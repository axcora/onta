## Wirte Articles

To write article you can access on `onta/content`

### Static Pages

If you want to create a static page such about and others , you can create new files and name it with your article example: `about.txt`

Then follow this frontmatter format
```

title: Write your title for SEO here
description: This is a description for your site
layout: page


Article goes here...
```

### Blog or Collections article

If you want to create a new collections, you can simply create a new folder and name it with your collections, example: `services`

Next you can create a new article `services/myservice1.txt`

And now you can write an article, follow this frontmatter format:

```
title: Blog title here...
description: A description here...
image: Your image path here...
date: 2026-02-13
category: Future
tags: AI, Automation, Zetta-Core


Write your article here...
```

After create a collections post, you need to create list page for your collections, so you can simply create a new file and name it with your collections folder, example `services.txt`

then follow this frontmatter format:

```
title: Our Services
description: A collection of articles about services
layout: collections-list
collection_title: New Article
collection_description: New Update from Axcora Zetta Core Articles.

```

### Other Pages

Next you can update your tags or categories Collections page

Open `tags.txt` or `categories.txt` and update it.
