# Onta SSG - Official Website
> **Axcora Zetta Core Concept**
> *Perl-Driven Static Site Architecture for Speed and Efficiency.*

This repository contains the source code and content for the official website of **Onta SSG**, accessible at [onta.axcora.com](https://onta.axcora.com). Built using the proprietary Axcora Zetta Core concept, this site demonstrates the power of ultra-lightweight static site generation.

## 🚀 Architecture
Onta SSG is designed for developers who value performance over complexity.
* **Engine:** Custom Perl-based Static Site Generator.
* **Concept:** JAMSTACK (JavaScript, APIs, and Markup).
* **Design:** Axcora Zetta Core Robotics Scheme (Automation-ready).
* **Deployment:** Automated via GitHub Actions.
* **Hosting:** GitHub Pages with Custom Domain.

## 🛠 Project Structure
* `/onta` - The core engine, configuration, and source content (.txt metadata).
* `/layout` - HTML templates (Main, Page, Collection-post, Partials).
* `/root` - Production-ready HTML files generated by the engine.

## ⚡ Performance Features
* **Zero Database:** No SQL overhead, ensuring 100% security and speed.
* **Clean Code:** Minimalist HTML/CSS output without bloat.
* **SEO Optimized:** Automatic meta tag injection and sitemap generation.
* **Zetta Core Integration:** Ready for API, AI, and robotics automation.
